# asp-net-examples
Examples for ASP.NET in .NET 1.1, IIS 5, and IE6

> These examples are only for education purposes. Since they lack of security **don't use in production!**

## For `asp4.soap.lan` the following other steps are required.

1. Create a `db` in Microsoft SQL Server 2005.
2. Then create a table `legends` with the following fields
- id, int, primary
- name, nchar(100)
- age, int
3. Fill the table with some random values.
4. Ensure the correct rights for IIS on the db, and table.

## For `asp5.soap.lan` give also write access to IIS.
